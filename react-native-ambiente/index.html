React Native - Configuração de ambiente

Abaixo estão descritos os passos para a configuração do ambiente React Native:

    EVENTUAL - PRIMEIRA VEZ:
  • Instalar o Chocolatey Software - https://chocolatey.org/install
  • Instalar a dependência do Chocolatey Software - No Power Shell digitar: choco install -y nodejs-lts microsoft-openjdk11
  • Instalar o Android Studio
  • Configurar o SDK no Android Studio
  • Configurador o emulador no Android Studio
  • Configure a variável de ambiente ANDROID_HOME
  • Adicionar ferramentas de plataforma ao Path
    PARA CADA PROJETO:
  • Executar o PowerShell como Administrador
  • Criar novo projeto: npx react-native@latest init nomeDoProjeto
  • Iniciar o Metro: npx react-native start
  • Iniciar o aplicativo: npx react-native run-android
